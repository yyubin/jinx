  </Card>
  <Card title="Plain SQL Output" icon="document">

  </Card>
  <Card title="Liquibase Support" icon="setting">
    Outputs Liquibase YAML as a secondary dialect for teams already using Liquibase for execution and tracking.
  </Card>
  <Card title="Extensible via SPI" icon="puzzle">
